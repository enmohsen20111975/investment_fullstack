# فهرس المهارات المعتمدة للمشروع

هذا الملف يعمل كذاكرة مشروع للمهارات المنسوخة من `D:\applications\skills` إلى المشروع.

## مكان المهارات داخل المشروع

- `docs/project_memory/skills/`

## المهارات المختارة ولماذا

1. **fastapi-pro**
   - الاستخدام: تحسين تصميم نقاط API، Dependency Injection، إدارة الاستجابات والأخطاء، الأداء.
   - المسار: `docs/project_memory/skills/fastapi-pro/SKILL.md`

2. **backend-architect**
   - الاستخدام: هندسة طبقات الخدمات، فصل المسؤوليات، قابلية التوسع، الجاهزية للويب والموبايل.
   - المسار: `docs/project_memory/skills/backend-architect/SKILL.md`

3. **api-design-principles**
   - الاستخدام: توحيد عقود API، التسمية، النسخ، التوافق مع عملاء متعددة (Frontend + Mobile).
   - المسار: `docs/project_memory/skills/api-design-principles/SKILL.md`

4. **risk-metrics-calculation**
   - الاستخدام: مقاييس المخاطر الاستثمارية (التذبذب، العائد المعدل بالمخاطر، السحب الأقصى...) لدعم القرار.
   - المسار: `docs/project_memory/skills/risk-metrics-calculation/SKILL.md`

5. **quant-analyst**
   - الاستخدام: نماذج التحليل الكمي، بناء استراتيجيات تقييم، دعم توصيات الذكاء الاصطناعي بمدخلات رقمية.
   - المسار: `docs/project_memory/skills/quant-analyst/SKILL.md`

6. **i18n-localization**
   - الاستخدام: اعتماد العربية كلغة أساسية في التسميات، الرسائل، وتجربة الاستخدام.
   - المسار: `docs/project_memory/skills/i18n-localization/SKILL.md`

7. **flutter-expert**
   - الاستخدام: ضمان توافق تصميم الـ API وتجارب الاستخدام مع تطبيق Flutter المستقبلي.
   - المسار: `docs/project_memory/skills/flutter-expert/SKILL.md`

8. **mobile-developer**
   - الاستخدام: تحسين نقاط النهاية وسيناريوهات الاستهلاك الخاصة بتطبيقات الجوال.
   - المسار: `docs/project_memory/skills/mobile-developer/SKILL.md`

## طريقة الاستخدام داخل المشروع

- عند تنفيذ أي ميزة جديدة، ارجع أولًا إلى المهارة المناسبة داخل `docs/project_memory/skills`.
- في متطلبات الاستثمار (الحسابات/الاستراتيجيات)، ابدأ بـ:
  - `risk-metrics-calculation`
  - `quant-analyst`
- في أي تغييرات API:
  - `fastapi-pro`
  - `api-design-principles`
- في واجهات المستخدم العربية وتطبيق الهاتف:
  - `i18n-localization`
  - `flutter-expert`
  - `mobile-developer`

## ملاحظة

تم اختيار هذه المهارات لأنها الأكثر ارتباطًا بمتطلباتك الحالية: بيانات حقيقية، دعم الاستثمار والتحليلات، واجهات عربية، وجاهزية backend لتطبيق ويب وموبايل.
